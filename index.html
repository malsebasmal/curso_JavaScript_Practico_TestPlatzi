Variables y operaciones

¿Qué es una variable y para qué sirve?

Una variable es un espacio que hemos guardado en memoria. Las variables sirven para almacenar valores, funciones y cual quiero otra cosa que nosotros queramos guardar en un lugar específico en memoria

¿Cuál es la diferencia entre declarar e inicializar una variable?

Declarar = Es separar el espacio en memoria donde se ubicará nuestra variable
inicializar = Es almacenar/asignar los elementos dentro de nuestra variable.

¿Cuál es la diferencia entre sumar números y concatenar strings?

La suma* de números es una operación matemática.
La concatenación* de string (cadenas de texto/texto) es juntar nuestros strings en uno solo.
Sin embargo, es importante aclarar que en JavasScript si se puede hacer la "suma" entre números y strings que son números. A eso se le llama coerción; explicación sencilla:
La transformación de cierto tipo de valores a otros ciertos tipo de valores.

¿Cuál operador me permite sumar o concatenar?

El operador que permite tal acción es el signo => +

Funciones

¿Que es una función?

Un función es un bloque de código que contiene ciertas instrucciones con un fin*. Estos también sirven para poder reutilizar estas mismas.

¿Cuando me sirve poner una función en mi código?

Cuando queremos realizar ciertas instrucciones, y que estas puedan ser reutilizadas.

¿Cuál es la diferencia entre parámetros y argumentos de una función?

Condicionales

¿Que es una condicional?

Es un bloque de código que únicamente se ejecutará si "x" condición se cumple, caso contrario no se ejecutará o en su defecto pasará otra cosa.

¿Qué tipos de condicionales existen en JavaScript y cuáles son sus diferencias?

Existen 2 condicionales principales. Las cuales son:
if / else = podemos tener diferentes expresiones para evaluar dependiente del caso, y generar tantos If o else como sea necesario.
switch = Este toma una sola expresión de entrada pero puede evaluar todos los casos como sea necesario de una forma más ordenada.

¿Puedo combinar funciones y condicionales?

Claro que si. Dentro de las mismas funciones puedes crear condiciones que se ejecuten cada vez que llamamos a nuestra función, o viceversa; podemos crear condicionales que contengan funciones, y que estas se ejecuten únicamente si cierta condición se cumple.

Ciclos

¿Qué es un ciclo?

Un ciclo es un bloque de código que itera determinadas veces. Es decir, algo que se repite varias veces. Podemos utilizarlo para recorrer objetos, arrays, o repetir un bloque de código varias veces.

¿Qué tipos de ciclos existen en JavaScript?

Existen tipos de ciclos:
For (Con todas sus variaciones) => For in, For of, .foreach
while
Do while

¿Qué es un ciclo infinito y por qué es un problema?

Los ciclos infinitos ocurren cuando no tenemos un "alto" en el código del loops. Es decir, cuando no determinamos cuando es que el loop debería detenerse.
Estos son un problema bastante grave, debido a que el loop nunca parara de repetirse, impidiendo que el usuario realice más acciones, y asu vez, este puede hacer que la página web se detenga por completo.

¿Puedo mezclar ciclos y condicionales?

Correcto. Esto nos ayudaría a que nuestro loop se repita, si únicamente la condición se cumple.

Listas

¿Qué es un array?

Un array en JavaScript es un valor de tipo objeto, el cual puede contener dentro de si, distintos tipos de valores.

¿Qué es un objeto?

Un objeto en JavaScript es un valor de tipo objeto, que también puede contener distintos tipos de valores dentro de él; pero la diferencia radica en que en el valor de tipo objeto, este tiene "claves" para cada uno de los tipos de valores que colocamos.

¿Cuándo es mejor usar objetos o arrays?

Varia mucho (como siempre) dependiendo del contexto en que estemos escribiendo el código.
Los array se utilizan para almacenar indefinida cantidad de tipos de valores, sin embargo, si es que necesitamos una plantilla para cierta funcionalidad de nuestro código, lo mejor será utilizar un objeto.

¿Puedo mezclar arrays con objetos o incluso objetos con arrays?

Correcto, dentro de un array puedes poner un valor de tipo objeto y dentro de un objeto puedes poner un array.
